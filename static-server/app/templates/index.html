PostgreSQL Static File Server


App to upload and download static files from PostgreSQL App through HTTP APIs.


To restore backup files in PostgreSQL database, select the backup path based on the PostgreSQL user you are using.
The default user is edge@siemens.com.
Then press Upload and wait until the file is loaded.
To restore the uploaded file, from the PGAdmin interface go to Tools -> Restore and then browse for the uploaded file.
